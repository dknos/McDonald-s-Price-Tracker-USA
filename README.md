# McDonald's USA Price Tracker

**Compare real menu prices across 14,000+ McDonald's locations in all 50 states.**

Find the cheapest McDonald's near you, browse state-wide price charts, and build an order to compare totals across stores.

**Live site: https://dknos.github.io/McDonald-s-Price-Tracker-USA/**

---

## How to Use

### Find Stores & Order
1. Enter your **ZIP code** and select a search radius
2. Click **"Find Stores & Order"**
3. Browse the menu by category (Burgers, Chicken, Breakfast, Drinks, etc.)
4. Tap items to add them to your cart
5. Click **"Compare Stores"** to see which nearby location has the lowest total

### Browse Prices by State
1. Click **"Browse Prices by State"** from the home screen
2. Pick any US state (or international market)
3. View the full price chart — every store as a row, every menu item as a column
4. Click any column header to sort stores by that item's price (cheapest to most expensive)
5. Use the ZIP code filter to narrow down to stores near a specific location
6. Use the search bar or category dropdown to find specific items

### Reading the Price Chart
- **Green cells** = cheaper, **Red cells** = more expensive (relative to other stores)
- **MIN / AVG / MAX** summary rows at the top show the price range for each item
- Items tagged **DEAL** or **PROMO** are flagged — these usually can't be combined with other promotions
- Use the highlight dropdown to switch between gradient view, cheapest-only, or priciest-only
